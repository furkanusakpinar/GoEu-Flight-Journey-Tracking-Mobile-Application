## 📱 About the App

This application is developed for users to track their flights and travel plans with a modern interface.  
It offers an enriched experience with real-time data, animations, and dynamic content.

---

## ✨ Key Features

✈️ **Flight Management**: Add, edit, and delete trips

⏳ **Live Countdown**: Day / hour / minute / second-based tracking

🌦️ **Instant Weather**: Live data for departure and arrival cities

📅 **Weather Forecast**: Custom forecast system tailored to travel dates

🎥 **Dynamic Background**: City-specific videos and images

🔐 **Admin Panel**: Secure login and management system

🔄 **Real-Time Sync**: Instant data updates via Firebase

💎 **Premium UI/UX**: Blur effects, animations, and modern design

---

## 🛠️ Technology Stack

**Framework**: React Native (Expo)

**Database**: Firebase Firestore

**API**: OpenWeather API, Pixabay API

**Video & UI**: Expo Video, Expo Blur

**Animation**: React Native Animated & SVG
